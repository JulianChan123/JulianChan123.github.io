Honey Nets

Un HoneyNet es una red de señuelo diseñada para atraer a los atacantes y estudiar sus métodos y tácticas. Es esencialmente un conjunto de sistemas de computadoras que simulan ser una red real, pero están configurados con el propósito de atraer y detectar actividades maliciosas.

Tunneling

El tunneling es la encapsulación de un protocolo de red dentro de otro. Es una técnica que permite el transporte de datos de una red a través de otra, creando un "túnel" virtual a través de la red de transporte. Este método se utiliza para enviar datos de un extremo a otro de una red que podría no ser capaz de admitir el protocolo de la capa superior directamente.

Internet of Things

La Internet de las cosas (IoT) describe la red de objetos físicos ("cosas") que llevan incorporados sensores, software y otras tecnologías con el fin de conectarse e intercambiar datos con otros dispositivos y sistemas a través de Internet. Estos dispositivos van desde objetos domésticos comunes hasta herramientas industriales sofisticadas.
